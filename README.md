# BrenkBot — Telegram-бот для @BrenkDesign

**Живая, многогранная и немного опасная.**  
Бот создавался для пользования аудиторией канала, для взаимодествия и развлечения.

### Особенности
- Полная анонимная переписка с Brenk  
- AI-CHAT на Groq (Llama 3.3 70B) — мгновенные ответы  
- **Три режима общения**:  
  → **Обычный** — тёплый, душевный, меланхоличный  
  → **Флирт** — игривый, кокетливый, с искрами  
  → **Безумный 18+** — откровенный, страстный, без цензуры (с подтверждением возраста)  
- Поддержка фото, видео, стикеров, документов  
- Анонимные комментарии к постам канала  
- Ответы от Brenk с сохранением медиа
