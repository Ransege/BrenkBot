# BrenkBot — Telegram-бот для @BrenkDesign

**Живая, многогранная и немного опасная.**  
Бот создавался для пользования аудиторией канала, для взаимодествия и развлечения.

### Особенности
- Полная анонимная переписка с Brenk  
- AI-CHAT на Groq (Llama 3.3 70B) — мгновенные ответы  
- **Три режима общения**:  
  → **Обычный** — тёплый, душевный, меланхоличный  
  → **Флирт** — игривый, кокетливый, с искрами  
  → **Безумный 18+** — откровенный, страстный, без цензуры (с подтверждением возраста)  
- Поддержка фото, видео, стикеров, документов  
- Анонимные комментарии к постам канала  
- Ответы от Brenk с сохранением медиа  
- Всё работает 24/7, без задержек

### Как запустить (локально)

# 1. Клонируем репозиторий
git clone https://github.com/твой_ник/brenk-bot.git
cd brenk-bot

# 2. Устанавливаем зависимости
pip install -r requirements.txt

# 3. Создаём .env файл
cp .env.example .env
# Открываем и вставляем свои ключи:
  BOT_TOKEN=твой_токен
  GROQ_API_KEY=gsk_твой_ключ
  OWNER_ID=твой_id

# 4. Запускаем!
python main.py

#Требования

→ Python 3.9+
→ Токен от @BotFather
→ Ключ от Groq (бесплатно до 10к токенов) 
**ВАЖНО""
Groq не работает на территории РФ (там уже каждый знает, что делать с ограничениями)
